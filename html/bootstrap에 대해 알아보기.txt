BootStrap
css/javascript기반 웹 프레임워크이다. 
반응형 웹 지원해주고, 아이콘, 버튼 등을 무료로 지원해준다. 브라우저 호환 가능.
단점: 성능이 다소 떨어진다. 
웹프레임워크: 웹을 만드는 재료들의 모음 

부트스트랩 CDN 을 복사해서 헤드 태그에 붙여 넣어야 부트스트랩의 여러 테마들을 사용할 수 있다. 
jquery 사용이 필요하기 때문에 https://code.jquery.com/ 여기로 이동하여
제일 최신버전 jQuery3.x 를 복사하여 짜던 코드의 헤드 부분에 붙여넣는다.
---------------------여기까지 부트스트랩 진행하면 제공하는걸 사용 가능하다.-----------
부트스트랩 사용하기
콘테이너: 양쪽에 여백을 만들기
홈페이지 오른쪽에 버튼 타입 선택하면 여러 버튼의 종류들을 볼 수 있고, 사용하고 싶은 버튼의 코드를 복사하여 붙여넣으면 된다.
